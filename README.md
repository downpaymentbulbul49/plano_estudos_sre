## 📚 About plano_estudos_sre

This application offers a clear learning guide for anyone interested in Site Reliability Engineering (SRE). It organizes study paths by skill level and focuses on free, online content, mainly in Portuguese. The guide covers technical topics like reliability, automation, monitoring, incident response, and platform engineering.

The plan walks you through learning from basic concepts up to advanced architecture and strategy. It suits those aiming to grow within SRE roles:

- Junior SRE
- Mid-level SRE
- Senior SRE
- Specialist/Architect SRE

The guide also groups topics into four main areas:

- SRE Fundamentals
- Essential Technologies
- Observability
- Pipelines and CI/CD

It includes widely used tools like Linux and various cloud and automation technologies.
